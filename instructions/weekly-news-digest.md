# Weekly Points & Miles News Digest

## What It Does

Every Wednesday at 8am UTC, the script:
1. Scrapes 7 aviation/loyalty news blogs for articles published in the last 7 days
2. Filters out credit card content and irrelevant articles
3. Sends the filtered articles to Claude, which selects the 10 most relevant and writes a community-ready post
4. Emails the formatted post to hello@estelletramaine.com

The email contains a post title and the full community post ready to copy and paste.

## Topics Covered

- Alliance changes (airlines joining or leaving alliances)
- Airline partnership and codeshare changes
- Points pricing changes and buy-points promotions
- Peak and off-peak award calendar changes
- Loyalty programme rule changes
- New ways to buy or earn points without a credit card
- International route additions and removals
- Status matching and challenge opportunities
- Fuel surcharge (YQ/YR) changes on award bookings
- Award seat availability changes

## Topics Excluded

- Credit card sign-up bonuses and credit card reviews
- US/UK domestic route news (unless it affects international connections)
- Hotel news unless directly tied to airline point transfers

## Post Format

Each email contains:
- A weekly post title (generated by Claude based on the week's top stories)
- Opening: "Happy Wednesday everyone!"
- A 2-3 sentence intro summary
- Up to 10 news items, each with:
  - Subheading (rewritten article title)
  - Bullet points (max 10 per item)
  - "Why this matters:" bullet (relevance to buying points / booking business class)
  - "Read more:" link (blog article URL, with a note if an official airline source exists)
- A closing call-to-action line

## Setup Requirements

### 1. Anthropic API Key (required)
The script uses the Claude API to generate the digest. You need an API key.

1. Go to console.anthropic.com/settings/keys
2. Create an API key
3. Add it to `.env`: `ANTHROPIC_API_KEY=your_key_here`
4. Add it as a GitHub Actions secret: repo Settings > Secrets and variables > Actions > New secret > `ANTHROPIC_API_KEY`

Cost: roughly $0.02-0.10 per weekly run depending on article volume.

### 2. GitHub Actions Secrets (already configured)
- `AUTOMATION_GMAIL` — the sending Gmail address
- `AUTOMATION_GMAIL_PASSWORD` — the Gmail App Password
- `ANTHROPIC_API_KEY` — add this one (see above)

```bash
python scripts/weekly_news_digest.py
```

Requires `ANTHROPIC_API_KEY` to be set in `.env`.

## Adding or Removing Sources

Edit the `RSS_SOURCES` list in `scripts/weekly_news_digest.py`. Each entry needs a `name` and a `url` pointing to the site's RSS feed (usually `sitename.com/feed/`).

## Adjusting Topics

To change what Claude focuses on, edit the `SYSTEM_PROMPT` variable in the script. The priorities list in the prompt controls what stories Claude selects and how it weights them.

## Troubleshooting

| Issue | Fix |
|-------|-----|
| "ANTHROPIC_API_KEY is not set" | Add the key to `.env` and GitHub Actions secrets |
| "No articles fetched" | One or more RSS feeds may be down — check URLs manually |
| Email not sending | Check `AUTOMATION_GMAIL` and `AUTOMATION_GMAIL_PASSWORD` in GitHub Actions secrets |
| Claude selects irrelevant articles | Refine the topic priorities in `SYSTEM_PROMPT` |

## Known Limitations

- The script links to the blog article, not the original airline/programme announcement. Claude will note where an official source likely exists, but you will need to click through to find the exact URL.
- AwardWallet's news feed URL may change — if it stops working, check awardwallet.com/blog for the current RSS link.
- Some sites may block automated requests intermittently. The script logs which sources failed so you can check manually if needed.
